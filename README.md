=# Parallel Computing Systems

Лабораторные работы по дисциплине **"Параллельные вычислительные системы"**.

### Стек технологий
- **C++** (100%)
- OpenMP
- MPI (в некоторых работах)

### Структура
Папки `pr0` — `pr6` — отдельные лабораторные работы.

### Смысл проекта
Изучение и практическое применение технологий параллельного программирования: распараллеливание циклов, работа с потоками, межпроцессное взаимодействие.

### Для чего
- Выполнение лабораторного практикума
- Освоение параллельного программирования
- Сравнение производительности последовательных и параллельных версий алгоритмов
